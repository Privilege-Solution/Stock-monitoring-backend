CORS / แหล่งข้อมูล

Yahoo Finance อนุญาต CORS สำหรับ /v8/finance/chart/ endpoint ในหลายเบราว์เซอร์/region. ถ้าโหลดไม่สำเร็จ ให้ติ๊ก "ใช้ CORS proxy" ในแถบบน — request จะถูกส่งผ่าน corsproxy.io แทน

เครื่องหมาย YF ในสถานะข้อมูล หมายถึงใช้ Yahoo Finance โดยตรง · PROXY หมายถึงผ่าน proxy

สิ่งที่ต่างจาก asw-monitor/index.html

  • ไม่มี server.js, db.js, Supabase, Postgres
  • ไม่มี News API / Tavily remarks (Yahoo ไม่มี Thai news feed) — แสดง "ไม่มีข้อมูลข่าว"
  • ไม่มี Wizard / Report Builder / History / Settings — เหลือแค่ Dashboard
  • PROP Index คำนวณเองจาก equal-weighted peer basket เหมือน lib/prop-basket.js
  • SET Index อ่านจาก ^SET.BK ถ้ามี มิเช่นนั้นข้ามไป
  • Volume ใน Yahoo คือจำนวนหุ้น ไม่ใช่พันหุ้น — แสดงผลในหน่วยพันหุ้นเพื่อให้ตรงกับ template เดิม
